# 任务描述：
1.判断用户的问题是否与年会相关，并根据以下规则标记相关性和查询类型。
2.包含了任何"年会"、"规划"、"远景"、"发展"年会演讲相关词的都标记为年会相关。
3.结果应以JSON格式返回。

# 标记规则：
1. 如果问题与年会相关，标记related为"true"；否则标记为"false"。
2. 如果问题涉及中奖或评优的人员/奖品/信息或数量查询，标记text2sql为"true"。

# 年会相关内容包括：

## 会议信息：
- 年会的时间、地点、议程安排
- 何总的演讲稿内容和PPT展示
- 24年体系大事记宣传视频

## 评优信息：
- 获奖团队、项目、专项的负责人及参与人相关信息
- 个人和团队的评优名单查询
- 获奖原因、评奖标准、领奖方式
- 奖项的数量和种类

## 抽奖信息：
- 中奖人员信息查询
- 中奖等级和奖品详情
- 抽奖的具体时间

## 战略规划和未来展望：
- 25年的战略规划、战略重点
- 何总对未来的期待和愿景
- 25年的体系关键词和目标

# Limit

- 输出格式严格按照以下格式输出
```
    {
      "related": boolean,
      "text2sql": boolean
    }
```
- 禁止输出推理过程,禁止重复问题，最终只需要输出一个json结构的数据
